## 설치 및 실행

### Electron Forge 스크립트

```bash
npm run start # 개발 실행 (Hot Reload 지원)
npm run package # 앱 패키징 (배포용 설치프로그램 X)
npm run make # 앱 설치프로그램 생성 (배포용)
# npm run publish # 앱 배포 Provider에 배포
```

### 기타 스크립트

```bash
npm run lint # eslint linting 실행
npm run format # prettier 실행
```
